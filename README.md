##### install
```bash
git clone https://github.com/jloiola/kowloon.git
cd kowloon
npm install
```

##### prod
```bash
npm start
```

##### dev
```bash
npm run dev
```

Highly recommend https://atom.io/ and https://atom.io/packages/language-marko for development
